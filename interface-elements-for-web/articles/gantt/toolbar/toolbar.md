The toolbar provides the following commands:

{|
|-
! Command
! Description
|-

| Undo
| Reverses the last action.
|-

| Redo
| Repeats the undone action.
|-

| Add Task 
| Adds a task.
|-

| Delete Task
| Deletes a task.
|-

| Collapse Tasks 
| Collapses all tasks.
|-

| Expand Tasks
| Expands all tasks.
|-

| Zoom In 
| Zooms in the chart.
| -

| Zoom In and Zoom Out
| Zooms out the chart.
| -

| Full Screen
| Switches the Gantt to full screen.
|}
