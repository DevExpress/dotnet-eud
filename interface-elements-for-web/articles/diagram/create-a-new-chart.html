Create a New Chart

This topic explains how to create a new chart and customize its settings.

Add a Shape

  1. Drag a shape from the Toolbox onto the page.

  2. Double click the shape to add/edit its text.

Connect Shapes

  1. Add another shape to the chart.

  2. Hover the mouse pointer over a shape to display its connection points.

  3. Hover a point, press the left mouse button, drag the connector to another shape's connection point, and release the mouse button. Now, if you move a shape, the connector stretches to it.

  4. Double click the connector to add text. You can use the mouse to move the text along the connector.

Place a Shape Into a Container

You can arrange shapes into collapsible horizontal and vertical containers.

Drag a shape over a container (this highlights the container) and drop the shape.

Resize a Shape

  1. Click a shape to select it.

  2. Drag the squares at the shape's borders and corners to resize the shape.

Insert an OrgChart Shape (Shape with Image)

  1. Drag a shape from the Org Chart toolbox group onto the page.

  2. Double click the shape to change its text.

  3. Right click the shape to invoke the context menu. Select the Insert Shape Image... command.

  4. Select the shape image and click OK.

To change or delete the image, select the corresponding command in the context menu.

Customize Shape and Connector Style Settings

  1. Select a shape, connector, or group of elements.

  2. Click the Properties button to invoke the Properties toolbar.

  3. Use the editors on the Style tab to customize style settings.
