# Connect to a MongoDB Data Source

The topics in this section describe the steps required to connect a report to a MongoDB instance. This task includes the following steps:

* [Specify Connection Properties](connect-to-a-mongo-db/specify-connection-properties.md)
* [Configure Queries](connect-to-a-mongo-db/configure-queries.md)
* [Select Data Fields](connect-to-a-mongo-db/select-data-fields.md)

> [!Note]
>  The **MongoDB.Driver** package should be installed in your project to supply MongoDB data at runtime.
