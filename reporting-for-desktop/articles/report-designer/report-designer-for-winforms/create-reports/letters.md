# Letters 

This tutorial describes the steps to create a mail merge report, in which data fields are embedded into a labelâ€™s text that is replaced with corresponding data values on preview or export.

3. Drop the [Rich Text](../use-report-elements/use-basic-report-controls/rich-text.md) or [Label](../use-report-elements/use-basic-report-controls/label.md) control from the [Toolbox](../report-designer-tools/ui-panels/toolbox.md) onto the [Detail](../introduction-to-banded-reports.md) band.
