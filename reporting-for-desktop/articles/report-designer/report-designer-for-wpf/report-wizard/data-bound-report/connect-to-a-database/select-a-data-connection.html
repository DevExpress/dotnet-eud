Select a Data Connection

On this page, you can choose whether to use one of the existing data connections or create a new one.

Click Next to proceed to the next wizard page. If you select one of the available connections from the list, go to the Customize the Query page. Otherwise, proceed to the Specify a Connection String page to create a custom connection string and manually specify its parameters.
