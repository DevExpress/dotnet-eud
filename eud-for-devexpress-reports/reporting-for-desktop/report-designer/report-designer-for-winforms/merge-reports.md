# Merge Reports

You may have report pages that do not fit within an entire report template in the following cases:

* [Title pages or custom pages at the end of the report param($match) $path = $match.Groups[1].Value; if ($path -notmatch '^https?://' -and $path -notmatch '^~/' -and $path -notmatch '^\.\./\.\./') { '](' + '../' + $path + '.md)' } else { $match.Value } ;
* [Charts within a table report param($match) $path = $match.Groups[1].Value; if ($path -notmatch '^https?://' -and $path -notmatch '^~/' -and $path -notmatch '^\.\./\.\./') { '](' + '../' + $path + '.md)' } else { $match.Value } ;

You can create pages in a separate report and merge them into your base report. This enables you to print and export merged pages as a single document, and preserve the original report page settings and orientation.
