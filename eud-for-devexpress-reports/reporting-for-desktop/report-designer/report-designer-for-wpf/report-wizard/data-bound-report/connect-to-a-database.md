The following data source types are supported:

* Microsoft SQL Server
* Microsoft Access 97
* Microsoft Access 2007
* Microsoft SQL Server CE
* Oracle
* Amazon Redshift
* Google BigQuery
* Teradata
* Firebird
* IBM DB2
* MySQL
* Pervasive PSQL
* PostgreSQL
* SAP HANA
* SAP Sybase Advantage
* SAP Sybase ASE
* SQLite
* VistaDB
* VistaDB5
* XML file
