# Scale Print Preview 

To scale a document, click the **Scale** ![Scale button icon](../../../images/img7267.png) button on the toolbar. The **Scaling** dialog is invoked.

![Scaling dialog window](../../../images/img7275.png)

* Set the required zoom percentage in the editor (for example, 100% or 125%).

* Set how many pages should fit across the viewer width in the editor (for example, 1 page wide or 2 pages wide).

Click **OK** to save changes and close the dialog.

> [!NOTE]
> If you're working with a document [loaded from file](../file-management/load-a-print-preview-from-a-file.md), you cannot use scaling.
